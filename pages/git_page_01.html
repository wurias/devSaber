CONFIGURAÇÕES


- git config --global user.name "seu-nome" (define seu nome de usuário)

- git config --global user.email "seu-email" (define seu email)

- git config --list (listar suas configurações)

- git config --global init.defaultBranch main (altera ramificação padrão para 'main')

- git config --get user.name (verifica usuário definido

- git config --get user.email (verifica email definido)
